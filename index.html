To broadcast a message to all investors, add a row to the Dispatches sheet with tag announcement. It will appear as the banner for all users on next load.
Sheet columns:
ID  |  Text  |  announcement  |  false  |  timestamp

To remove: delete the row or change the tag away from announcement.

📋 How to post a dispatch
Add a row to the Dispatches tab in your Google Sheet.
Columns: ID  |  Text  |  Tag  |  Pinned  |  Timestamp  |  Edited
Tag values: general, update, alert  ·  Pinned: true or false
Timestamp: Unix ms (e.g. paste =INT((NOW()-DATE(1970,1,1))*86400000))
Dashboard refreshes every 60s — or reload the page to see new posts immediately.
